# shiny-text
✨Shiny Text✨

See <https://mizdra.github.io/shiny-text>

## Keybind
- ``e`` or ``i``: Edit text (shiny-text)
- ``Espace`` or ``Enter``: Finish editing
- ``c``: Change contrast
- ``s``: Spin text


## shiny-image
If you drop image, the image is ✨shine✨!!
